# FastFood Ordering System

A modern Full Stack Fast Food Ordering System built using **ASP.NET Core 10 Web API** and **Angular** following **Clean Architecture** principles.

The application allows customers to browse food items, place orders, manage carts, submit reviews, and securely authenticate using JWT. It also provides an admin panel for managing food items, categories, users, and orders.

---

## Features

### Customer

- User Registration & Login
- JWT Authentication
- Browse Food Items
- Search Food
- Filter by Category
- View Food Details
- Shopping Cart
- Place Orders
- Order History
- Reviews & Ratings
- Profile Management

### Admin

- Dashboard
- Manage Categories
- Manage Food Items
- Upload Food Images
- Manage Orders
- Manage Users
- View Reviews
