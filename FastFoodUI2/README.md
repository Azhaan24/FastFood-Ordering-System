# Fast Food Ordering System

## Features

- JWT Authentication
- Role-Based Authorization
- Food Management
- Category Management
- Shopping Cart
- Checkout
- Order Tracking
- Admin Dashboard
- Image Upload
- Charts
- Responsive UI

## Tech Stack

### Frontend
- Angular 22
- Bootstrap 5
- Chart.js
- SweetAlert2

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- ASP.NET Identity
- JWT Authentication
